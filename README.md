## What is included?

- [Devenv](https://devenv.sh/) for development environment management.
- [uv](https://docs.astral.sh/uv/) for Python dependency management.
- [Taskfile](https://taskfile.dev/) as a task runner to simplify common development tasks.
- [Lefthook](https://lefthook.io/) for managing Git hooks.
- [Ruff](https://ruff.rs/) as a linter and code formatter.
- [Black](https://black.readthedocs.io/en/stable/) as a code formatter.
- [Pytest](https://docs.pytest.org/en/7.4.x/) as a testing framework.
- [MkDocs](https://www.mkdocs.org/) for project documentation.
- [GitHub Actions](https://docs.github.com/en/actions) for CI/CD.
- [Renovate](https://docs.renovatebot.com/) for dependency updates.

## üöÄ Getting Started

### Pre-Requisites

This template is built with [Copier](https://copier.readthedocs.io/en/stable/), a Python based project templating tool.

To install copier on your system, follow the instructions at [Copier Website](https://copier.readthedocs.io/en/stable/#installation)

### Usage

To create a new project using this template, run the following command:

```bash
copier copy gh:brpaz/copier-python /path/to/your/new/project
```

And answer the prompts to customize your new project.
